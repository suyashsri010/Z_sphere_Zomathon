Our project addresses the intelligence gap in current food delivery cart recommendations, where generic add-on suggestions often fail because they lack an understanding of the specific items already in a cart, the user’s personal history, and the temporal context of the order. The challenge is to build a system that can deliver highly relevant, context-aware suggestions at scale, specifically identifying incomplete meals and complementary items while managing a latency of less than 300ms. By utilizing a two-stage retrieval and ranking approach powered by LightGBM and LLM semantic embeddings, we aim to solve core issues like recommendation fatigue and cold-start scenarios. Ultimately, the system is designed to intelligently maximize Average Order Value (AOV) and add-on acceptance rates without compromising the user experience or cart-to-order conversion.
